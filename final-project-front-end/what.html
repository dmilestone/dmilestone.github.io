The Data

Data Source Information and Feedback...

Our data came from Inside Airbnb - Adding Data to the Debate.

  • listings.csv - containing 16 columns and 2441 rows of rental listing data
  • calendar.csv - containing 7 columns and 890,965 rows of booking data
  • reviews.csv - countaing 6 columns and 170,973 rows of reviews matching different listings
Insights

From matplotlib, plt.tight_layout() is a gamechanger!

Finding, cleaning, and understanding a dataset seems like the biggest challenge and most time consuming task.

Not having any training data was a challenge for sentiment analysis so we relied on third party library not centered around airbnb reviews.

To use this dataset for a reliable machine learning model, we'd recommend combining listing data from multiple cities to create a greater level of in predicting outcomes.
